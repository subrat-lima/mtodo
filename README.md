# mtodo

## about
mtodo is a minimal todo web application

## features
1. add/delete todo
2. save data to sqlite
3. login with google oauth
4. auto sync via websockets

## usage
```sh
# update dependency
uv sync

# set environment
. ./.venv/bin/activate

# run app
python app.py
```
